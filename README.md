<p align="center">
  Aplicativo de receitas e dietas personalizadas. O Nouri sugere receitas com base nas preferências alimentares e restrições dietéticas do usuário, com suporte a planejamento semanal de refeições e geração automática de lista de compras.
</p>

## ✨ Funcionalidades

- **Onboarding** com configuração de restrições alimentares e objetivos
- **Recomendações personalizadas** na tela inicial
- **Busca e filtros** por ingrediente, restrição e tipo de refeição
- **Visualização completa de receitas** com informações nutricionais e ajuste de porções
- **Plano semanal de refeições** com arrastar e soltar
- **Lista de compras** gerada automaticamente a partir do plano
- **Perfil** com receitas favoritas e preferências salvas

---

## 🚀 Como rodar o projeto

### Pré-requisito

- [Node.js](https://nodejs.org) v18+

### Instalação

```bash
git clone https://github.com/sunrhse/Nouri.git
cd Nouri
npm install
npm run dev
```

O projeto estará disponível em `http://localhost:5173`.
